1. You an inspect the cronjob using the following command:

   kubectl --namespace {{ .Release.Namespace }} get cronjob {{ include "pantheon-cron.fullname" . }}

   kubectl --namespace {{ .Release.Namespace }} create job --from=cronjob/{{ include "pantheon-cron.fullname" . }} manual-pantheon-cron
